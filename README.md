IDE utilizado -> 
  Apache NetBeans IDE 23.
  
Versión del lenguaje de programación utilizado. -> 
  JAVA JDK 23.

DBMS utilizado y su versión. ->
  SQL SERVER EXPRESS 2019 15.0.2000
  Microsoft SQL Server 2019 (RTM) - 15.0.2000.5 (X64) 
	Sep 24 2019 13:48:23 
	Copyright (C) 2019 Microsoft Corporation
	Express Edition (64-bit) on Windows 10 Enterprise Evaluation 10.0 <X64> (Build 22621: ) (Hypervisor)

Lista de pasos para correr su aplicación. ->
  1.- Instalación de servicio TOMCAT 10.1.49.
  2.- Instalación de servicios JAVA y SQLSERVER.
  3.- Ejecutar scripts de sql para levantar modelado de datos.
  4.- Pergar archivo ejercicio1.war en directorio de tomcat -> ApacheSoftware/tomcat10/webapps/
  4.1.- El servicio de tomcat despliega el proyecto en automático.
  5.- Acceder al aplicativo a través de la ruta -> localhost/ejercicio1/app/jsp/login/login.jsp
